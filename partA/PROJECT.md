# PROJECT.md

## Сэдэв
URL Shortener

## Товч тайлбар
Урт URL-ийг богино код болгон хөрвүүлдэг, дарсан тоог тоолдог, хугацаа дуусах боломжтой веб апп.

## Scope
### Хамрах
- URL богиносгох (random short code үүсгэх)
- Богино URL дээр дарахад анхны URL руу redirect хийх
- Click counter (хэдэн удаа дарсан тоолох)
- URL expiration (хугацаа тогтоох)
- Бүх URL-ийн жагсаалт харах

### Хамрахгүй
- Хэрэглэгчийн бүртгэл / нэвтрэх
- Custom short code
- Analytics dashboard
- QR code
